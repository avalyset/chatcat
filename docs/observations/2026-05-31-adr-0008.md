# 2026-05-31 — ADR 0008 og baseline-normalisert reward

Baseline-normalisert reward. 0007 ga flatt signal — agenten hadde ingen
gradient å klatre på. 0008-grepet: normaliser mot baseline. Løste
flatheten, agenten nådde baseline-linjen. Men holdt den ikke —
climb-then-slide, robust over tre kjøringer. Poenget: å løse flathet
skapte et nytt problem som så ut som ustabilitet. Signalet fantes nå;
spørsmålet ble om det kunne holdes. (Run-artefaktene på /tmp ble senere
vasket — kjøre-datoene er kun i commit og ADR-tekst.)
